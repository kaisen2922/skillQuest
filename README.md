# 🎮 SkillQuest — Gamified Learning Platform for Tech Skills

> Turn coding education into an immersive adventure.  
> Learn skills, complete missions, earn XP, climb leaderboards, and level up like a true Code Knight.

---

## 🚀 Overview

SkillQuest is a gamified learning platform designed to make learning technical skills fun, addictive, and rewarding.

Instead of boring courses and static dashboards, SkillQuest transforms education into an RPG-inspired world where users progress through skill tracks, solve coding challenges, unlock badges, and compete globally.

Built for the **Adivya 2.0 Developer Hackathon**.

## ⚔️ Gamification Engine

### XP Rewards
- Complete Lessons → +10 XP
- Finish Quiz → +20 XP
- Solve Coding Challenge → +50 XP
- Daily Login → +5 XP

### Rank Progression
- Rookie
- Developer
- Senior Dev
- Architect
- Legend

### Achievements / Badges
- First Lesson
- 7-Day Streak
- Quiz Master
- Speed Coder
- Full Stack Explorer
- 100 Problems Solved

## 🔥 Daily Challenge System

- New challenge every midnight
